# 🌱ctf-study
My CTF study notes, scripts and reports
这个仓库用于记录我的学习过程，包括计算机网络、Linux、Python、渗透测试、CTF 题目等。  
目的：  
- 见证自己的成长  
- 在未来回望时能看见当初的脚印  
- 给学习增添一点仪式感与成就感

## 📂 仓库结构
```
ctf-study/
├─ README.md # 仓库说明
├─ studyplans/ # 学习计划
├─ notes/ # 学习笔记（Markdown/PDF）
├─ scripts/ # 学习过程中的脚本
├─ reports/ # 题解、实验报告
├─ resources/ # 截图、参考资料
```
